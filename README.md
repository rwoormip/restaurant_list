# 我的餐廳清單
使用 Node.js + Express

## 功能
- 使用者可以在首頁瀏覽所有餐廳
- 使用者可以點擊任一餐廳查看餐廳的詳細資訊
- 使用者可以透過餐廳中、英文名稱、類別來搜尋餐廳

## 環境建置
- Node.js
- Node Package manager (npm)
- Nodemon
- Express v4.18.2
- Express-handlebars v3.0.0
